🌑 O Projeto

O CloudWare é uma infraestrutura de segurança distribuída. Ele atua na linha de frente da proteção digital através de uma Extensão de Navegador conectada a um core de Inteligência de Ameaças desenvolvido em FastAPI.

O sistema não apenas verifica arquivos maliciosos, mas orquestra dados de múltiplas fontes, oferecendo um assistente de IA para suporte em tempo real e um feed de notícias automatizado.


⚙️ Engenharia de Back-end

Como o foco deste projeto é demonstrar proficiência em Back-end e APIs, a arquitetura foi desenhada seguindo padrões de mercado:

    API Design: Endpoints RESTful de alta performance utilizando processamento assíncrono (asyncio).

    Security Layer: Sistema de autenticação via API Keys e proteção contra injeção de dados.

    Integrações de Terceiros: Consumo via Requests/HTTPX de motores de detecção globais para análise heurística.

    Cache & Performance: Otimização de requisições para reduzir a latência na resposta da extensão.
